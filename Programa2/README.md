## Cómo ejecutar el programa (Python)

### 1. Requisitos
- Tener **Python 3** instalado

### 2. Pasos
- Abrir la terminal
- Ubicarse en la carpeta `Python`
- Ejecutar:
```bash
python ordenamientos.py
```
- Ingresar el tamaño del arreglo cuando se solicite
- Elegir el algoritmo de ordenamiento del menú (1-6)

## Cómo ejecutar el programa (Java)

### 1. Requisitos
- Tener instalado el JDK de Java

### 2. Pasos
- Abrir la terminal
- Ubicarse en la carpeta `Java`:
```bash
cd Programa2/Java
```
- Compilar:
```bash
javac Ordenamientos.java
```
- Ejecutar:
```bash
java Ordenamientos
```
- Ingresar el tamaño del arreglo y elegir el algoritmo del menú (1-6)

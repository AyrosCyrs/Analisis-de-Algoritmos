# Análisis de Algoritmos

Programas desarrollados para la materia de Análisis de Algoritmos (Facultad de Ciencias, UNAM). Cada programa está implementado en **Java y Python**, cubriendo temas de programación orientada a objetos, algoritmos de ordenamiento y teoría de gráficas.

## 📂 Contenido

| Programa | Tema | Lenguajes |
|---|---|---|
| [Programa1](./Programa1) | Modelado de objetos y recorridos lineales | Java, Python |
| [Programa2](./Programa2) | Algoritmos de ordenamiento (Insertion, Selection, Bubble, Counting, Merge, Quick Sort) | Java, Python |
| [Programa3](./Programa3) | Núcleo de una digráfica acíclica | Java |

Cada carpeta tiene su propio README con instrucciones detalladas para ejecutar el programa correspondiente.
